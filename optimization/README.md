# Optimization

Combinatorial and continuous optimization problems formulated as BQM or CQM and solved on the Dynex network.

## Examples

| Notebook | Description |
|----------|-------------|
| [Portfolio Optimization](Dynex_Portfolio_Optimisation.ipynb) | Markowitz mean-variance portfolio optimization with real market data |
| [Number Partitioning](quantum_number_partitioning.ipynb) | Partition a set of integers into two equal-sum subsets |
| [Graph Partitioning](quantum_graph_partitioning.ipynb) | Divide a graph into balanced partitions minimizing cut edges |
| [Job Sequencing](quantum_job_sequencing.ipynb) | Schedule jobs to minimize total makespan |
| [Set Cover](quantum_set_cover.ipynb) | Find the minimum collection of sets covering all elements |
| [Vertex Cover](quantum_vertex_cover.ipynb) | Minimum vertex cover — smallest set of vertices touching all edges |
| [K-Means Clustering](quantum_kmeans_clustering.ipynb) | Quantum-assisted data clustering |
| [Binary Integer Linear Programming](quantum_BILP.ipynb) | General BILP encoded as QUBO |
| [Aircraft Loading](aircraft-loading-optim.ipynb) | Cargo loading optimization for aircraft weight and balance |
| [Traffic Optimization](TrafficOptimizationCQMBUG.ipynb) | City traffic routing using CQM |
