## MAX-CUT G70 Result

The G70 graph has 800 nodes and 4,694 edges. Dynex achieved a MAX-CUT value of **9578**, surpassing previously published results from both classical and quantum approaches.

See [G70_dynex.ipynb](G70_dynex.ipynb) for the full experiment and validation.
